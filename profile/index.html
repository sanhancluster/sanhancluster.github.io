Research

NewCluster simulation

I'm a builder of NewCluster simulation, cosmological hydrodynamic zoom-in simulation that aims to reproduce a galaxy cluster with very high resolution (~70 pc).

Check out more

Improving astrophysical simulation code RAMSES

RAMSES uses MPI-based multiprocessing, which is inefficient at very large number of cores (~10,000). We implemented OpenMP in RAMSES to construct hybrid paralleism, which greatly increases scalability of the code.

Velocity dispersion profiles of galaxies

We used high-resolution zoom-in simulation of galaxies, NewHorizon, to measure the velocity dispersion profiles of galaixes and compare them with IFU observations.
